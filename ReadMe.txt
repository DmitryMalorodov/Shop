В данной работе содержатся:
- Тест-Сьюты в папке "Test-Suites"
- Фреймворк и автотесты написанные на языке Java (в папке "Shop")

Проводилось тестирование сайта - http://automationpractice.com/

Для запуска автотестов необходимо:
- скачать драйвер - "chromedriver"
- установить путь до вашего драйвера в переменной "chromeDriverPath" в классе "Resources.java" (класс находится -  \\Shop\src\test\java\resources\Resources.java)
